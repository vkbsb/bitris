RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py


STORY:
      "Hello Program, Welcome to the 8-bit computer. " \
      "your task is to return the binary representation of the number asked for." \
      "The bits are color coded and numbered for your convenience." \
      "You have limited time to return the required number before the request times out." \
      "Too many timeouts or errors, and you will be Terminated."


HOW TO PLAY THE GAME:
Create the binary representation of the number shown on top "Return: X"
Where X is the number, say the top says "Return: 10" you are supposed
to create the binary representation of "10" in the falling bits. Left click
on the bits you want to make zero. Careful, your toggles cannot be un-done.
